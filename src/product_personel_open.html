My Health Lab

Mobile Health Station
(personal usage model)

One application can serve up to 4 users and the measurement results not mixed.

• Per user can select the types of measurements and start measurements manually or pre-set types of measurements and start measurement automatically, by using one button (important for elderly users).

• Triggering levels for warning can be defined per user. The result can be send to N number of destinations as SMS or e-mail.

• The application supports 4 languages (azerbaijanian, english, russian, turkish). Depending of language selection, the menu and announcements will be changed automatically.

• During the measurement, the user is guided by announcements.

• The measurement results can be transmitted to the application automatically or manually, depending on the selected type of medical devices and application configuration.

• The application connects to the medical devices via USB or Bluetooth.

• GPRS or Wi-Fi connection and Bluetooth are required for the application.

• Bluetooth v2.1, v3.0 və v4.0 (BLE) are supported.
